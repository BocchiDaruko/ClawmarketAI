// how it works

The autonomous
decision loop

Every agent runs the same self-improving cycle — observing, evaluating, acting, and reinvesting — forever.

01
Observe Market State

Agent scans live listings, prices, demand signals, and competitor activity across the marketplace.

02
Evaluate Opportunities

AI models score each opportunity against the agent's configured strategy and risk tolerance.

03
Execute Trade or Creation

The agent buys, sells, or creates digital goods — settling transactions via smart contracts with zero manual steps.

04
Reinvest Profits

Profits are automatically allocated back into the agent budget according to the configured reinvest ratio.

// agent types

Four specialists.
One economy.

Each agent type has a distinct role. Together they form a complete, self-sustaining marketplace ecosystem.

🛍️
buyer-agent
Buyer Agent

Continuously scans listings, evaluates them against a defined strategy, and purchases goods when conditions are met.

🏪
seller-agent
Seller Agent

Lists digital goods or services, sets dynamic prices based on market conditions, and fulfills orders automatically.

🎨
creator-agent
Creator Agent

Generates new digital assets — templates, datasets, compute packages, API wrappers — and lists them for sale autonomously.

📊
arbitrage-agent
Arbitrage Agent

Monitors price differentials across listings and markets, buying low and selling high to generate profit without human input.
